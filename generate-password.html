Generate Admin Password Hash

Use this tool to generate a secure SHA-256 hash for your partner dashboard password.

⚠️ Keep this page private! Do not share the hash or deploy this file to production.

How to Update Your Password:

  1. Enter your desired password above and click "Generate Hash"
  2. Copy the generated hash
  3. Open partner-dashboard/index.html
  4. Find the line with ADMIN_PASSWORD_HASH (around line 517)
  5. Replace the hash value with your new hash
  6. Open supabase/functions/broadcast-notification/index.ts
  7. Find the line with ADMIN_PASSWORD_HASH (line 4)
  8. Replace the hash value with the same hash
  9. Deploy the function: npx supabase functions deploy broadcast-notification
  10. Deploy the dashboard to your hosting (Vercel, etc.)
